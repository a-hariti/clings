# 05 Input Parsing

Learn how to read and parse user input safely in C. This chapter focuses on
buffer sizes, conversion errors, and avoiding undefined behavior.

Topics:
- `fgets` for bounded input
- Parsing with `sscanf`
- Handling invalid input and partial reads

Related chapters:
- **04_string_literals**: String basics and null termination.
- **10_functions**: Function signatures for parsing helpers.
